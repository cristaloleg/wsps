### How to start

Install RabbitMQ:
```
brew install rabbitmq
```

Update PATH after installation
```
PATH=$PATH:/usr/local/sbin
```

Start RabbitMQ:
```
rabbitmq-server
```

Web view:
```
http://localhost:15672/#/
```
